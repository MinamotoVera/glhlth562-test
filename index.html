Project overview

This minimum viable product (MVP) retrieves global health indicator data directly from the World Bank API, cleans it in R, visualizes trends across countries, and produces a short plain-language interpretation. It is a first working version of the product proposed in the course proposal.

What this page does

  • Pulls data from the World Bank Open Data API
  • Compares life expectancy at birth across selected countries
  • Displays an interactive plot
  • Generates a concise interpretation based on the most recent available values

Automatically generated interpretation

Code 
top_country <- latest_data$country[1]
top_value <- latest_data$value[1]
top_year <- latest_data$year[1]

bottom_country <- latest_data$country[nrow(latest_data)]
bottom_value <- latest_data$value[nrow(latest_data)]

change_tbl <- wb_data |>
  group_by(country) |>
  summarise(
    first_year = min(year, na.rm = TRUE),
    last_year = max(year, na.rm = TRUE),
    first_value = value[year == first_year][1],
    last_value = value[year == last_year][1],
    change = last_value - first_value,
    .groups = "drop"
  ) |>
  arrange(desc(change))

largest_gain_country <- change_tbl$country[1]
largest_gain_value <- change_tbl$change[1]

cat(glue(
  'Using the most recent available World Bank data, **{top_country}** had the highest life expectancy among the selected countries at **{round(top_value, 1)} years** in **{top_year}**, while **{bottom_country}** had the lowest at **{round(bottom_value, 1)} years**. From **{start_year}** to **{end_year}**, **{largest_gain_country}** showed the largest overall increase, with life expectancy rising by approximately **{round(largest_gain_value, 1)} years**. This MVP demonstrates the core pipeline proposed for the final project: API retrieval, data cleaning, visualization, and automated interpretation.'
))
Using the most recent available World Bank data, **United States** had the highest life expectancy among the selected countries at **78.4 years** in **2023**, while **Brazil** had the lowest at **75.8 years**. From **2000** to **2023**, **Brazil** showed the largest overall increase, with life expectancy rising by approximately **6.3 years**. This MVP demonstrates the core pipeline proposed for the final project: API retrieval, data cleaning, visualization, and automated interpretation.

Notes for next iteration

This MVP is intentionally simple so that the pipeline works end to end. The next version can extend the proposal by adding:

  1. User-selected countries and indicators
  2. Additional World Bank health indicators
  3. A true GenAI summary step using Gemini or OpenAI API
  4. More comparative views and richer analytical components
